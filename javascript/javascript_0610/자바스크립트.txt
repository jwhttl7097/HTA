자바의 메소드
	접근제한자 반환타입 메소드명(매개변수타입 매개변수이름) {
		수행문;
		return 반환값;
	}
	접근제한자 void 메소드명(매개변수타입 매개변수이름) {
		수행문;
	}
	
	변수 : 기본자형값, 객체
	매개변수 : 기본자형값, 객체
	반환값 : 기본자형값, 객체

	*메소드를 변수에 담을 수 없다.
	*메소드를 매개변수로 전달할 수 없다.
	*메소드를 반환값을 사용할 수 없다.

자바스크립트의 함수
	function 함수명(매개변수명) {		// 접근제한자 없음, 반환타입 없음, 매개변수타입 없음
		수행문;	
		return 반환값;		// 이 함수를 실행하면 반환값에 해당하는 값을 반환받는다.
	}
	function 함수명 (매개변수명) {
		수행문;			// 
	}
	* 함수를 변수에 담을 수 있다.
	* 함수를 매개변수로 전달할 수 있다.
	* 함수를 반환값으로 사용할 수 있다.
